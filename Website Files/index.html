Frequently Asked Questions

Is Goldstrap safe to use?
Yes. Goldstrap is engineered with sophisticated structural safety integration providing a 99% ban-proof environment. Zero bans have been recorded to date across all users.
What is FastFlag Editor?
FastFlag Editor lets you take full control of internal client engine variables and experiment with advanced feature configurations. You can adjust rendering, performance, and UI settings that are normally hidden.
Does it work on macOS or Linux?
Currently Goldstrap is Windows-only as it natively integrates with the Windows Roblox client. macOS and Linux support is being explored for future releases.
Is Goldstrap open source?
Yes! Goldstrap is fully open source and forked from Fishstrap. You can view the source code, contribute, or report issues on our GitHub repository.
Can I use multi-instance to play multiple accounts?
Yes. The multi-instance selection feature allows you to launch and manage multiple Roblox client instances concurrently, each with their own separate configurations and channel deployments.
